    COOL AXSPOINT

    • Jul 2011 - June 2012
    • Role: Software Engineer

    The AXS-One Viewer is a powerful, user-friendly application that allows you to access documents online. The AXS-One Server is designed to manage large amounts of information and store it in a pre-processed, compressed state with indexing on both magnetic and other media. This allows for fast index searching of pre-indexed information as well as general text searching. The AXS-One Viewer can overlay the data stored by the AXS-One Central Server with forms, for a look and feel similar to the original document. In addition, the forms have the ability to work with rich graphics. The forms can also be printed with the data allowing you to avoid having to use custom pre-printed paper forms. Information can be exported in a variety of formats and can be integrated with several popular desktop packages, such as Microsoft Word and Microsoft Excel. The AXS-One Central Server provides full security at both the user and document level. There is an audit trail for monitoring many of these functions.
